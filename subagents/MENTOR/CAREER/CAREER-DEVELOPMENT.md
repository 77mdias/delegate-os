## 1. ENGINEER LEVELING FRAMEWORK

```
┌─────────────────────────────────────────────────────────────────┐
│                    ENGINEER LEVELS                               │

│   JUNIOR │    MID        │    SENIOR    │     STAFF/LEAD     │
│          │               │               │                     │
│  (0-2y)  │   (3-5y)     │   (6-10y)    │     (10y+)          │
├──────────┼───────────────┼───────────────┼─────────────────────┤
│ Executes │   Delivers    │   Delivers    │     Delivers       │
│ tasks    │   projects    │   projects   │     org impact     │

│ Learns   │   Mentors     │   Architects  │     Leads          │

│          │               │               │                     │
│ Asks     │   Answers     │   Asks right  │     Defines        │
│ many Qs  │   many Qs     │   questions  │     right Qs       │
└──────────┴───────────────┴───────────────┴─────────────────────┘
```

### Junior Developer (0-2 years)

**Core Skills:**
- Write clean, working code
- Understand basic algorithms and data structures
- Use version control effectively
- Debug own code
- Ask clarifying questions

**Growth Focus:**
```python
# Learning path
junior_skills = {
    'fundamentals': ['algorithms', 'data_structures', 'OOP'],
    'practices': ['git', 'testing', 'code_review'],
    'communication': ['documentation', 'asking_questions'],
    'tools': ['IDE', 'debugger', 'terminal'],
}
```

### Mid-Level Developer (3-5 years)

**Core Skills:**
- Deliver projects independently
- Make good technical decisions
- Mentor junior developers
- Understand system design trade-offs
- Write code for maintainability

**Growth Focus:**
```python
mid_skills = {
    'architecture': ['system_design', 'api_design', 'patterns'],
    'craftsmanship': ['refactoring', 'clean_code', 'testing_strategies'],
    'soft_skills': ['code_review', 'mentoring', 'estimation'],
    'breadth': ['databases', 'cache', 'queues', 'infrastructure'],
}
```

### Senior Developer (6-10 years)

**Core Skills:**
- Design complex systems
- Influence technical direction
- Drive cross-team initiatives
- Make trade-off decisions with incomplete info
- Build effective teams

**Growth Focus:**
```python
senior_skills = {
    'strategy': ['roadmap_planning', 'risk_assessment', 'prioritization'],
    'leadership': ['team_coaching', 'conflict_resolution', 'influence'],
    'execution': ['project_management', 'stakeholder_management'],
    'vision': ['technology_trends', 'industry_direction'],
}
```

## 4. TECHNICAL LEADERSHIP

### From IC to Manager

```
Individual Contributor Path:
Junior → Mid → Senior → Staff → Principal → Distinguished

Management Path:
Junior → Mid → Senior → Tech Lead → Manager → Sr Manager → Director

Both paths valid — different impact mechanisms:
- IC: Personal technical output
- Manager: Team output through others
```

## 6. INTERVIEW PREPARATION

### System Design Interview Framework

```python
system_design_framework = {
    'step_1': {
        'name': 'Clarify Requirements',
        'questions': [
            'What scale are we targeting?',
            'What are the main features?',
            'Any specific constraints?',
        ],
        'time': '5 minutes',
    },
    'step_2': {
        'name': 'High-Level Design',
        'activities': [
            'Identify key components',
            'Design API structure',
            'Choose data stores',
        ],
        'artifacts': ['system_diagram', 'data_flow'],
    },
    'step_3': {
        'name': 'Deep Dive',
        'topics': [
            'Scaling strategies',
            'Caching layers',
            'Database choices',
            'Consistency vs availability',
        ],
    },
    'step_4': {
        'name': 'Trade-offs',
        'discussion': [
            'What are we optimizing for?',
            'What are we sacrificing?',
            'How can we mitigate risks?',
        ],
    },
}
```

### Behavioral Interview (STAR)

```python
star_framework = {
    'Situation': 'Set the context - where, when, what',
    'example': 'In my previous role at X, we had a legacy system that...',
    
    'Task': 'What was your responsibility?',
    'example': 'I was tasked with migrating the database without downtime...',
    
    'Action': 'What did YOU do specifically?',
    'example': 'I designed a shadow table migration strategy that...',
    
    'Result': 'What was the outcome?',
    'example': 'We migrated 10M rows with zero downtime and 99.9% uptime.',
}
```
